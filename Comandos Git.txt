git pull
puxa os arquivos do repositório

git add --all (ou botão direito > stage)
prepara os arquivos para o commit

git status
verifica o estado dos arquivos

git reset arquivo
reseta o status do arquivo,

git commit -m "mensagem" (ou aquela caixa de diálogo do vsc)
cria um commit com os arquivos que estão "staged"

git push
manda pro repositório os commits locais
